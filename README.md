# WB Parser

Асинхронный парсер товаров с Wildberries по поисковому запросу  
Сервис собирает до ~90% товаров за счёт разбиения по ценовым диапазонам.

## 🚀 Возможности

- 🔍 Парсинг товаров по поисковому запросу
- ⚡ Асинхронная загрузка (httpx + asyncio)
- 📦 Сбор полной информации о товаре:
  - ссылка
  - название
  - цена
  - описание
  - изображения
  - характеристики
  - размер
  - продавец
  - остатки
  - рейтинг
  - количество отзывов
- 🧠 Умное разбиение диапазонов цен (обход лимита WB ~1000 товаров)
- 🎯 Фильтрация:
  - цена ≤ 10 000 ₽
  - рейтинг ≥ 4.5
  - страна производства: Россия
- 💾 Сохранение в Excel:
  - `wb_all.xlsx` — все товары по запросу
  - `wb_filtered.xlsx` — отфильтрованные по заданному условию

## 🛠️ Установка

Клонирование репозитория:
```bash
git clone https://github.com/your_username/wb-parser.git
cd wb-parser
```
Установка зависимостей:
```bash
pip install -r requirements.txt
```
Запуск парсера:
```bash
python parser.py
```

## ⚙️ Настройки

При необходимости в коде можно изменить:

Поисковый запрос:
```python
QUERY = "пальто из натуральной шерсти"
```
Фильтр:
```python
if price <= 10000 and rating and rating >= 4.5 and self.is_russia(features):
```
